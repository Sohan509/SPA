Greensock animation

Greensock Animation API (GSAP) is an animation library that helps you create performant animations. It can be used safely to create animations without jank, and it's the only animation library that handles SVG animations seamlessly.

NodeJs

Node.js is an open source development platform for executing JavaScript code server-side. Node.js is intended to run on a dedicated HTTP server with one process at a time. Node.js applications are event-based and run asynchronously.

Angular.io

AngularJS is a structural framework for dynamic web apps. It lets to use HTML extend HTML's syntax to express your application's components clearly and succinctly. AngularJS is what HTML would have been, had it been designed for applications.

asp.net core
